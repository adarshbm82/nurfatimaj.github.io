Multiple Imputation: Variable Selection

Simple simulation exercise

Last updated on Oct 17, 2019 3 min read research

The choice of imputation model in a multiple imputation setting is a very frustrating topic. One of the most popular recommendations in the literature is to always include an outcome variable when imputing a regressor to make sure that their covariance is not attenuated. In fact, variable choice recommendations sound like a kitchen-sink approach: include everything that could predict the imputed variable and missingness mechanism.

Our project is interested in the effect of intelligence score on college degree attainment, controlling for family background. College degree variable has missing observations. The dataset also has current earnings variable, which could be a good predictor of university degree. But it also depends on person’s intelligence and could affect the main parameter of interest.

Simulation

Context

I am setting up the simulation exercise as close as possible to our actual goal. There is a binary outcome variable with missing information and three complete continuous regressors.

  • Outcome variable: probability of having a university degree ($uni$)
  • Regressors:
    • IQ score ($iq$)
    • Family wealth ($wealth$)
    • Future income ($fincome$)

The main analysis model is $$\begin{equation}\Pr(uni) = F(\beta_0 + \beta_1 uni + \beta_2 wealth + \varepsilon)\end{equation}$$ and our main paramater of interest is $\beta_1$. The aim of this simulation exercise is to figure out if including or excluding $fincome$ into imputation model affects estimator of $\beta_1$.

Setup

I am using R package mice to run a multiple imputation. The simulation parameters are set up as follows

mean_x <- c(0, 0) # vector of means of uni and wealth
covar_x <- matrix(c(1, 0.2, 0.2, 1), 2, 2) # covar matrix of uni and wealth
beta_1 <- 0.1     # true \beta_1
beta_2 <- 0.05    # true \beta_2
alpha_0 <- 5      # intercept in log wage model
alpha_1 <- 0.15   # coefficient of uni in log wage model
alpha_2 <- 0.1    # coefficient of iq in log wage model
N <- 10000        # number of observations
p_miss <- 0.7     # probability of missing uni
M <- 70           # number of imputations

I have generated the university dummy using logit model. I create two different datasets under two missingness mechanisms: missing completely at random (MCAR) and missing non-randomly (MNAR).

data <- mvrnorm(n = N, mu = mean_x, Sigma = covar_x) %>%
  as.data.frame() %>% select(iq = V1, wealth = V2)
data <- data %>%
  mutate(p_uni = 1/(1 + exp(-beta_1*iq - beta_2*wealth)),
         uni = rbinom(N, 1, p_uni),
         lwage = alpha_0 + alpha_1*uni + alpha_2*iq + rnorm(N))
df_mcar <- data %>%
  mutate(u = runif(N),
         uni = ifelse(u <= p_miss, NA, uni),
         uni = as.factor(uni)) %>%
  select(-p_uni, -u)
df_mnar <- data %>%
  mutate(uni = ifelse(p_uni <= p_miss, NA, uni),
         uni = as.factor(uni))
